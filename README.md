# Number Analyzer (Python)

This is a simple command-line Python program that analyzes a number.

## Features

* Count total digits
* Find sum of digits
* Count even and odd digits
* Find largest digit
* Find smallest digit
* Reverse the number

## How to Run

1. Make sure Python is installed
2. Run the file:
>>bash
python your_file_name.py


>>Example
Enter a number: 1234

Digits: 4
Sum: 10
Even: 2
Odd: 2
Largest digit: 4
Smallest digit: 1
Reverse: 4321
